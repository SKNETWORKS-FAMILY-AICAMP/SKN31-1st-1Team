## 수집 데이터
### 1. 수집 데이터 출처
 - 중고차 매물 및 지점 정보 : K-Car(중고차 판매 사이트) 플랫폼
 - FAQ(자주 묻는 질문) 정보 : 현대 및 기아자동차 공식 홈페이지

### 2. 데이터 수집 목적
 - 차량 매물 정보 : 브랜드, 모델, 연식, 가격 등 상세 필터링 기능을 구현하고, 가격대별 매물 수와 연식별 시세 추이를 분석하여 사용자에게 객관적인 시장 정보 제공을 위함
 - 지점 정보 : 전국의 K-Car 지점 위치를 시각화하여 사용자가 오프라인 매장을 쉽게 찾을 수 있도록 지원하기 위함
 - FAQ 데이터 : 제조사별로 흩어져 있는 중고차 관련 주요 질문과 답변을 통합하여, 사용자가 궁금한 점을 한곳에서 쉽고 빠르게 검색할 수 있는 서비스를 제공하기 위함

### 3. 데이터 설명
#### - kcar_cars_raw.csv
  • **주요 포함 정보(컬럼)** : 브랜드, 모델, 연식, 가격, 주행거리  
  • **설명** : K-Car에서 크롤링한 중고차 실매물의 원천 데이터
#### - kcar_cars_cleaned.csv
  • **주요 포함 정보(컬럼)** : kcar_cars_raw.csv의 전처리 완료된 차량 상세 정보  
  • **설명** : 분석 및 DB 삽입을 위해 정제된 데이터(시세 및 매물 분석용)
#### - kcar_centerscsv
  • **주요 포함 정보(컬럼)** : 지점명, 위치 정보(위경도 등)  
  • **설명** : 전국 K-Car 지점의 주소, 전화번호, 위경도를 담고 있는 데이터
#### - hyundai_faq.csv
  • **주요 포함 정보(컬럼)** : 질문(Question), 답변(Answer)  
  • **설명** : 현대자동차 공식 홈페이지의 자주 묻는 질문 데이터
#### - kia_faq.csv
  • **주요 포함 정보(컬럼)** : 질문(Question), 답변(Answer)  
  • **설명** : 기아자동차 공식 홈페이지의 자주 묻는 질문 데이터
